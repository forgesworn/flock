# Flock compatibility vectors v1

This directory keeps the Flock app's JavaScript/Kotlin interoperability vectors.
The canonical public module manifest and protocol vectors live in
`forgesworn/flock-kit`; this consumer suite runs the native pipeline against the
immutable `@forgesworn/flock` dependency.

The JSON files cover the cross-language JavaScript/Kotlin boundary:

- `vectors.json` freezes key derivation, group IDs, geohashes, encrypted beacon
  decryptability, NIP-44 and NIP-59 wrapping.
- `radar-vectors.json` freezes the pure radar guidance rules.
- `kotlin-wraps.json` is generated by `npm run test:native` and consumed by
  `verify-kotlin.test.ts` for the reverse Kotlin-to-JavaScript direction. It is
  intentionally not committed.

Run:

```sh
npm test
npm run typecheck
npm run build
npm run test:native
```

Regenerate committed JSON only for an intentional protocol change:

```sh
npm run gen:vectors
```
